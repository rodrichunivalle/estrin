ESTRIN
Módulo de Inventarios de La Estrella


| Acerca del Sistema |


ESTRIN es un software simple de gestión de inventario creado para cumplir con los requisitos de La Estrella. El software es capaz de realizar un seguimiento del stock entrante y saliente con una factura generada dinámicamente. Cada operador del Sistema cuenta con un perfil donde pueden ver el historial de sus transacciones con la empresa.


| Facultades |


  • Gráfico interactivo para mostrar el stock disponible actualmente
  • Se mantienen detalles del stock, como el stock disponible actual
  • Las transacciones entrantes se realizan a través de compras
  • Las transacciones salientes se realizan a través de ventas
  • Cada transacción realizada a través del software se registra
  • El Comprobante se genera dinámicamente y se puede imprimir
  • Se mantienen perfiles separados para los proveedores
